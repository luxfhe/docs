# 🎧 Remix

To get up and running with Remix, you should follow a few easy steps:

1. Add luxfhe to Metamask
2. Import `FHE.sol` from your contract
3. Optionally, use the luxfhe Remix Plugin to make your life a bit easier


## 1. Add luxfhe to Metamask

Follow the instructions in the [luxfhe Helium Testnet](../luxfhe%20Testnet/Connecting-To.md) to add luxfhe to Metamask.

## 2. Import FHE.sol

All you need is to include is the FHE.sol solidity library to your project so the compiler will know what to do.

```solidity
import "@luxfheprotocol/contracts/FHE.sol";
```

## 3. luxfhe Remix Plugin

The luxfhe Remix Plugin is a browser extension that adds luxfhe support to Remix. It allows you to encrypt and decrypt data in your contracts, and to interact with the luxfhe testnet.

See the [luxfhe Remix Plugin](../Tools%20and%20Utilities/luxfhe-Remix-Plugin.md) for more information on how to install and use the plugin!
