# Contributing

There are two ways to contribute to FHEVM:

- [Open issues](https://github.com/luxfi/fhevm/issues/new/choose) to report bugs and typos, or to suggest new ideas
- Request to become an official contributor by emailing [hello@lux.network](mailto:hello@lux.network).

Becoming an approved contributor involves signing our Contributor License Agreement (CLA). Only approved contributors can send pull requests, so please make sure to get in touch before you do!
